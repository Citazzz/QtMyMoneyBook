## 一、已实现的功能

### 1. 用户注册与登录（已完成）

- 用户名/密码注册与登录，通过 QStackedWidget 在登录和注册页面间切换
- 密码使用 SHA-256 加盐哈希存储，不保存明文
- 输入验证：用户名至少 3 个字符，密码至少 4 个字符，注册需两次密码一致
- 注册成功后自动创建 1 个默认账户和 15 个默认收支分类

**对应文件**：`LoginDialog.h` / `LoginDialog.cpp`

### 2. 账单管理（已完成）

- 支持收支记录的增加、删除、修改，按时间倒序查看
- 每条记录包含：日期、类型（收入/支出）、分类、金额、备注
- 顶部实时汇总显示总收入（绿色）、总支出（红色）、结余（蓝色）
- 金额列按类型着色，选中行后启用修改/删除按钮
- 支持将全部账单导出为 CSV 文件

**对应文件**：`billwidget.h` / `billwidget.cpp`

### 3. 分类管理（已完成）

- 支持自定义收支分类的添加、修改、删除
- 注册时自动预置 5 个收入分类（工资、奖金、兼职、投资、其他收入）和 10 个支出分类（餐饮、交通、购物、住房、娱乐、医疗、教育、通讯、日用、其他支出）
- 收入/支出分类在表格中以不同颜色区分
- 切换账单类型时分类下拉框自动联动过滤

**对应文件**：`categorymanagerwidget.h` / `categorymanagerwidget.cpp`

### 4. 预算管理（已完成）

- 按月份设置月度支出预算金额
- 实时统计当月已支出金额，计算剩余预算
- 进度条分三级变色预警：低于 80% 显示绿色，80%~100% 橙色预警，超过 100% 红色超支
- 账单变更后预算状态自动刷新（通过 `billsChanged` 信号联动）

**对应文件**：`budgetwidget.h` / `budgetmanager.cpp`

### 5. 多条件搜索筛选（已完成）

- 支持四个维度的组合查询：关键字模糊搜索备注、收支类型筛选、具体分类筛选、日期范围筛选
- 搜索结果底部自动汇总：显示收入和支出合计以及总结余
- 汇总行以灰色背景高亮，视觉上区分于数据行
- 重置按钮一键清空所有筛选条件

**对应文件**：`searchfilterwidget.h` / `searchfilterwidget.cpp`

### 6. 数据可视化（已完成）

- **饼图**：按分类展示收入/支出金额占比，切片上显示分类名称和百分比，使用 15 色调色板确保各分类颜色区分
- **折线图**：展示每日收支变化趋势，X 轴为日期（-45° 倾斜标签避免重叠），Y 轴自动适配数据范围
- 支持切换收支类型和自定义统计时间范围，图表切换时自动刷新
- 使用 Qt Charts 模块，图表具有入场动画效果

**对应文件**：`visualizationwidget.h` / `visualizationwidget.cpp`

### 7. 全局主题样式（已完成）

- 统一的现代简约平面设计风格，蓝色（#5B7FFF）为主色调
- 覆盖 QWidget、QTabWidget、QGroupBox、QPushButton、QLineEdit、QTableWidget、QProgressBar、QScrollBar 等全部控件的样式
- 通过 `main.cpp` 中的 `APP_THEME` 常量统一管理

**对应文件**：`main.cpp`

### 8. 数据持久化（已完成）

- 基于 SQLite 本地数据库（`mymoney_data.db`），包含 4 张表：users、accounts、categories、bills、budgets
- 外键约束 + ON DELETE CASCADE 保证数据完整性
- 账单增删改时自动同步更新账户余额

**对应文件**：`databaseManager.h` / `databaseManager.cpp`

### 核心类一览

| 类名 | 继承 | 角色 | 核心功能 |
|------|------|------|---------|
| `databaseManager` | ——（单例） | 数据访问层 | 用户认证 / 账单CRUD / 分类CRUD / 预算管理 / 统计查询 / CSV导出 |
| `LoginDialog` | QDialog | 登录与注册 | SHA-256 密码验证 / QStackedWidget 双页切换 / 注册后自动初始化 |
| `MainWindow` | QMainWindow | 主窗口框架 | QTabWidget 管理 5 个 Tab / 信号槽联动（账单→预算刷新） |
| `BillWidget` | QWidget | 账单管理页 | 收支流水增删改 / 类型-分类联动过滤 / 汇总统计 / CSV导出 |
| `CategoryManagerWidget` | QWidget | 分类管理页 | 收入/支出分类增删改 / 15 个默认分类预置 / 弹窗编辑 |
| `BudgetWidget` | QWidget | 预算管理页 | 月度预算设置 / QProgressBar 进度 + 三级变色预警（80%/100%） |
| `SearchFilterWidget` | QWidget | 搜索筛选页 | 关键字+类型+分类+日期四维组合查询 / 结果汇总行 / 一键重置 |
| `VisualizationWidget` | QWidget | 数据可视化页 | 饼图（分类占比）/ 折线图（日趋势）/ 时间范围 + 类型切换 |
| `AccountManagerWidget` | QWidget | 账户管理页 ⚠ | 多账户增删改查（已编码，待集成） |

> ⚠ `AccountManagerWidget` 代码已完成但未加入编译、未挂载到主界面、数据库层 API 待补充。

### 类协作关系

```
main.cpp
  ├── LoginDialog ──→ databaseManager（登录验证 + 注册初始化）
  └── MainWindow
        ├── BillWidget ──→ databaseManager（账单 CRUD + CSV 导出）
        │     └── 信号 billsChanged() ──→ BudgetWidget.refresh()
        ├── CategoryManagerWidget ──→ databaseManager（分类 CRUD）
        ├── BudgetWidget ──→ databaseManager（预算设置 + 统计查询）
        ├── SearchFilterWidget ──→ databaseManager（多条件组合查询）
        └── VisualizationWidget ──→ databaseManager（分类/日统计数据）
```

**设计要点**：
- **单例模式**：`databaseManager` 全局唯一，各 Widget 通过 `instance()` 访问
- **信号槽解耦**：Widget 之间不直接通信，通过 `MainWindow` 转发信号
- **三层架构**：表现层（Widget）→ 数据库接口层（databaseManager）→ 存储层（SQLite）
