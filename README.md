# 个人记账系统 (MyMoneyBook)

基于 Qt6 的桌面端个人财务管理软件，采用现代简约平面设计风格。

每个用户拥有一个独立账户，所有收支记录自动汇总到该账户。

## 环境要求

- Qt 6.11+ (需要 Widgets、Sql、Charts 模块)
- CMake 3.16+
- MinGW / MSVC / GCC 编译器

## 快速开始

```bash
mkdir build && cd build
cmake ..
cmake --build .
./MyMoneyBook
```

或直接在 Qt Creator 中打开 `CMakeLists.txt` 编译运行。

## 功能说明

### 1. 注册与登录

首次使用需注册账号：

- 用户名至少 3 个字符，密码至少 4 个字符
- 密码使用 SHA-256 加盐哈希存储，不保存明文
- 注册成功后自动创建默认账户和 15 个默认收支分类

### 2. 账单管理

| 操作 | 说明 |
|------|------|
| 添加 | 选择类型、日期、分类，填写金额和备注，点击添加 |
| 修改 | 点击列表中的记录，修改表单内容后点击修改 |
| 删除 | 选中记录点击删除，确认后移除 |
| 导出 | 点击底部「导出CSV」，可将全部账单导出为 CSV 文件 |

- 顶部实时显示总收入、总支出和结余
- 收入金额绿色显示，支出金额红色显示

### 3. 分类管理

支持完全自定义收支分类，包括新增、修改、删除：

- 默认预置 5 个收入分类：工资、奖金、兼职、投资、其他收入
- 默认预置 10 个支出分类：餐饮、交通、购物、住房、娱乐、医疗、教育、通讯、日用、其他支出
- 每个用户独立管理自己的分类，互不影响

### 4. 预算管理

- 按月份设置支出预算金额
- 进度条实时显示当月支出占比
- 使用率低于 80% 显示绿色、80%~100% 橙色预警、超过 100% 红色超支

### 5. 搜索筛选

支持多条件组合查询：

- 关键字模糊搜索备注
- 按收支类型筛选
- 按具体分类筛选
- 按日期范围筛选
- 结果底部自动汇总收入、支出和结余

### 6. 数据可视化

- **饼图**：按分类展示收入/支出金额占比
- **折线图**：展示每日收支变化趋势
- 可自由切换收支类型和统计时间范围

## 数据存储

所有数据保存在运行目录下的 `mymoney_data.db`（SQLite），包含以下数据表：

| 表名 | 说明 |
|------|------|
| `users` | 用户账号（SHA-256 加盐哈希密码） |
| `accounts` | 用户默认账户（系统自动管理） |
| `categories` | 收支分类 |
| `bills` | 账单记录 |
| `budgets` | 月度预算 |

备份或迁移数据只需复制该 `.db` 文件即可。
